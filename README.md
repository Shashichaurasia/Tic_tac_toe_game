# Tic_tac_toe_game
This is a web-based implementation of the classic Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game allows two players (X and O) to take turns marking spaces in a 3Ã—3 grid. The objective is to align three marks in a row, column, or diagonal to win the game. The game includes a responsive design, interactive gameplay, and features like automatic win detection, game reset functionality, and visual effects for enhanced user experience.
# Features
1:- Responsive and user-friendly interface
2:- Two-player gameplay
3:- Automatic win detection and highlighting
4:- Game reset functionality
5:- Visual indicators for player turns and game status
